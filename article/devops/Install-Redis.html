Redis安装 ​

系统：Ubuntu Server 22.04 LTS 64bit

版本：Redis

下载Redis ​

shell
wget https://github.com/redis/redis/archive/7.0.11.tar.gz

安装编译环境

shell
apt install make gcc pkg-config

解压

shell
tar -xvf 7.0.11.tar.gz

编译

shell
make -j4
# 当make编译中断时，深度清理 
make distclean

安装

shell
# 默认安装
make install
# 指定安装目录
make PREFIX=/data/middware/redis install

修改Redis.conf

shell
# 从源码复制redis.conf
cp redis.conf /data/middware/redis/bin/redis.conf
#1、设置守护进程
daemonize yes

#2、关闭 监控
supervised no

#3、注释bind ip支持远程范围
bind 0.0.0.0

#4、禁用保护模式
protected-mode no

#5、设置密码
requirepass 123456

#6、启动redis
/data/redis/bin/redis-server /data/source/redis/redis-7.0.11/redis.conf
